839 
840 --We want to have type-checking on the functions put into the vtables. Type checking
841 happens only with functions, not macros, so we need a type_check function for every
842 vtable.
843 
844 --Only once in your codebase, you'll need to #define Declare_type_checking_fns to
845 actually define the type checking function. Everywhere else, the function is merely
846 declared.
847 
848 --All other uses point to having a macro, such as using __VA_ARGS__ to allow any sort
849 of inputs to the hash.
850 
851 --We want to have such a macro for every vtable. That means that we need a macro
852 to write macros. We can't do that with C macros, so this file uses m4 macros to
853 generate C macros.
854 
855 --After the m4 definition of make_vtab_fns, each new vtable requires a typedef, a hash
856 definition, and a call to make_vtab_fns to do the rest.
857 */

867  (m1)->p ? (size_t)(m1)->p*33 : \
868  (m1)->draw ? (size_t)(m1)->draw*33*27 \
