842 This declares the vtable macros for each procedure that uses the mechanism.
843 
844 --We want to have type-checking on the functions put into the vtables. Type checking
845 happens only with functions, not macros, so we need a type_check function for every
846 vtable.
847 
848 --Only once in your codebase, you'll need to #define Declare_type_checking_fns to
849 actually define the type checking function. Everywhere else, the function is merely
850 declared.
851 
852 --All other uses point to having a macro, such as using __VA_ARGS__ to allow any sort
853 of inputs to the hash.
854 
855 --We want to have such a macro for every vtable. That means that we need a macro
856 to write macros. We can't do that with C macros. Thus, this file uses m4 macros to
857 generate C macros.
858 
859 --After the m4 definition of make_vtab_fns, each new vtable requires a typedef, a hash
860 definition, and a call to make_vtab_fns to do the rest.
861 */
